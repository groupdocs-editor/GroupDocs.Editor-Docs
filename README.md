# GroupDocs.Editor - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Editor](https://docs.groupdocs.com/editor/) static documentation.

## About GroupDocs.Editor

[GroupDocs.Editor](https://products.groupdocs.com/editor/) is a document editing SDK that enables you to edit documents in various formats including Word, Excel, PowerPoint, and other popular file types.

Available for .NET, Java, and Node.js.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/editor/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
